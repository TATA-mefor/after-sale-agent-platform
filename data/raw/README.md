# Raw Demo Datasets

Place downloaded public source datasets under this directory for local processing only.

Expected layout:

```text
data/raw/orders/
data/raw/chinese_reviews/
data/raw/clothing_reviews/
```

The repository keeps only this README and `.gitkeep` placeholders. Raw CSV, XLSX, JSON, archive, and parquet files are
ignored because they are large, externally sourced, and should remain reproducible from the original public download
locations.

Do not commit raw datasets, personal filesystem paths, credentials, or private customer data.
